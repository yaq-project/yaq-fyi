configuration ¶

aoi_binning (string) default: 1x1
Option to bin pixels both horizontally and vertically. \ '1x1', '2x2', '4x4', or '8x8'. Default is no binning ('1x1').

aoi_height (['null', 'int']) default: None
Specify the number of pixel rows to acquire. \ Can be no larger than `sensor_height` - (`aoi_top` - 1). \ Defaults to use all rows starting from `aoi_top`. \

aoi_left (['null', 'int']) default: None
Specify left-most pixel column of the AOI. \ Defaults to the full left of the camera sensor (column 1). \

aoi_top (['null', 'int']) default: None
Specify the top-most pixel row of the AOI. \ Defaults to the top of the camera sensor (1). \

aoi_width (['null', 'int']) default: None
Specify the number of pixel columns to acquire. \ Can be no larger than `sensor_width` - (`aoi_left` - 1). \ Defaults to use all columns starting from `aoi_left`. \

enable (boolean) default: True
Disable this daemon. The kind entry-point will not attempt to start this daemon.
from is-daemon

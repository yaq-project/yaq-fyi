configuration ¶

aoi_hbin (int) default: 1

aoi_height (['null', 'int']) default: None
Specify the pixel height. \ Can be no larger than sensor_height - (aoi_top - 1). \ Default is the full height of camera. \ Units of super-pixels. \

aoi_left (['null', 'int']) default: None
Specify left most coordinate of AOI. \ Defaults to the full left of the sensor (1). \ Units of pixels. \

aoi_top (['null', 'int']) default: None
Specify top coordinate (units of pixels). \ Defaults to the top of the camera sensor (1). \ Units of pixels. \

aoi_vbin (int) default: 1

aoi_width (['null', 'int']) default: None
Specify width of AOI, in units of super-pixels. \ Defaults to the full width of sensor.

enable (boolean) default: True
Disable this daemon. The kind entry-point will not attempt to start this daemon.
from is-daemon
